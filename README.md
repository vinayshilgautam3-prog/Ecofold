# EcoFold - Eco-Friendly Paper Bags Website

## Description
EcoFold is a static HTML website designed for selling sustainable paper bags. It features a clean, eco-themed design with product listings, contact info, and responsive layout. Perfect for promoting biodegradable alternatives to plastic. Live demo: [Your GitHub Pages URL, e.g., https://yourusername.github.io/ecofold-site]

## Features
- **Homepage**: Hero section with eco-messaging and product highlights.
- **Products Page**: Listings for standard and custom paper bags with prices and buy buttons (static for now; integrate with e-commerce platforms like Shopify for real sales).
- **About and Contact**: Information on sustainability and business details.
- **Responsive Design**: Works on desktop and mobile.
- **SEO Basics**: Includes meta tags for search visibility (e.g., keywords like "eco-friendly paper bags").

## Installation and Setup
1. **Clone the Repository**: Run `git clone https://github.com/yourusername/ecofold-site.git` (or download ZIP from GitHub).
2. **Open Locally**: Double-click `index.html` in your browser to preview.
3. **Host Online**: 
   - For GitHub Pages: Go to Repository Settings > Pages > Enable on main branch. Site will be live at `https://yourusername.github.io/ecofold-site`.
   - For other hosts: Upload files to your web server (e.g., via FTP to public_html).
4. **Customize**: Edit `index.html` in a text editor, then commit changes to GitHub for auto-updates.
